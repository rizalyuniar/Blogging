## Getting Started

### Installation

- Clone This Front End Repo

```
git clone https://github.com/rizalyuniar/Blogging.git
```

- Go To Folder Repo

```
cd Blogging
```

- Install Module

```
npm install
```

- Type ` npm run dev` To Start Website
- Type ` npm run production` To Start Production
